Git

  • git init - создает новый репозиторий Git.
  • git clone URL - копирует репозиторий с удаленного сервера.
  • git status - показывает состояние репозитория (измененные файлы и т.д.).
  • git add filename - добавляет файл в индекс для будущего коммита.
  • git commit -m "message" - сохраняет изменения с сообщением.
  • git branch название ветки - создает новую ветку.
  • git checkout название ветки - переключается на другую ветку.
  • git push - отправляет коммиты на удаленный сервер.
  • git pull - получает изменения с удаленного сервера.
  • git log - показывает историю коммитов.
